<h2>ChatRoomManager</h2>
<p>Both a NodeJs and JavaScript Class that can manage chat rooms. This package requires <a href="http://socket.io/">Socket.IO</a> to work properly</p>

<h3>Supported Browsers:</h3>

<ul>
  <li>IE 9+</li>
  <li>Mozilla Firefox</li>
  <li>Google Chrome</li>
  <li>Apple Safari</li>
  <li>Opera</li>
</ul>
